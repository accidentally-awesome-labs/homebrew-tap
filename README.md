# homebrew-tap

Homebrew tap for packages published by [accidentally-awesome-labs](https://github.com/accidentally-awesome-labs).

## Install

```bash
brew tap accidentally-awesome-labs/tap
brew install --cask runnerkit
```

Fully qualified install also works:

```bash
brew install --cask accidentally-awesome-labs/tap/runnerkit
```

This tap is updated automatically by GoReleaser on every `vX.Y.Z` tag pushed
to `accidentally-awesome-labs/runnerkit`. Casks live in `Casks/`.
